Introduction

Gridicarus (Grid Icarus) is a CSS framework built around CSS Grids. Its main purpose is to build and prototype grid-based websites quickly and easily. And while it's still in development, it allows grid-based websites (like this one) to be created almost instantaneously without any setup. Take a look at the source code for this page to see how Gridicarus is being used!

Getting Started

To get started with Gridicarus, use either npm:

npm install gridicarus

Or the CDN link:

<link rel="stylesheet" href="https://unpkg.com/gridicarus@0.1.2/dist/gridicarus.min.css">

Grid-Parent

Defining a grid layout.

To create a new grid, simply use the .grid-parent class:

<div class="grid-parent">
	<div class="grid-child #1 box-blue">
		This is some content!
	</div>
</div>
This is some content!

This will create a 1:1 grid that densifies its children.

If a parent grid wants to structure its children, it can do so with these options:

  • .h-fill(1-9) - Horizontally fills empty grid cells.
  • .v-fill(1-9) - Vertically fills empty grid cells.
  • .h-gap-size(1-9) - Creates a horizontal gap for each child.
  • .v-gap-size(1-9) - Creates a vertical gap for each child.

Here's a basic example of this website's layout using the options above:

<div class="grid-parent">

  <div class="grid-child #1 h-span2 box-lime">
    Header
  </div>
  <div class="grid-child #2 h-span2 box-blue"> 
    Body
  </div>
  <div class="grid-child #4 box-grey">
    Footer Left
  </div>
  <div class="grid-child #4 box-silver">
    Footer Right
  </div>

</div>
Header
Body
Footer Left
Footer Right

Notice that we used .h-span2 to dictate that wanted 2 evenly sized rows. We have to assign .h-span(#) to every child in the grid, unless we have an even number of children. In which case we just assign .h-span(#) to each child we'd like to resize.
